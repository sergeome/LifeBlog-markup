# LifeBlog-markup

Responsive blog Markup example.

What were used:

HTML
* HTML5
* "Shema.org" Microdata
* Snippets

CSS
* SCSS preprocessor
* BEM methodology
* Responsive design with media query

Javascript
* JQuery library
* Modular approach
